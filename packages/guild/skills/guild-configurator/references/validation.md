# validation

Use esta referência quando a mudança precisar provar que a config continua válida.

## Conteúdo mínimo

- validar sintaxe JSONC/JSON
- checar presença e tipo dos campos obrigatórios
- conferir referências cruzadas entre agents, categories, prompts e skills
- confirmar que a alteração não remove intenção original

## Regra rápida

Se houver erro de schema, dado ausente ou saída inesperada, leia isto antes de concluir.
