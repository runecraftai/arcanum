# config-map

Use esta referência para mapear intenção -> chave -> efeito.

## Conteúdo mínimo

- `agents`: overrides de agentes existentes
- `custom_agents`: agentes novos e seus vínculos
- `categories`: agrupamento e roteamento
- `prompt`, `prompt_file`, `prompt_append`: origem e composição do prompt
- `skills`: skills associadas ao agente

## Regra rápida

Se a dúvida for “onde isso vai na config?”, consulte este mapa antes de editar.
