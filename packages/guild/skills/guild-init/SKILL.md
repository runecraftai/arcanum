# guild-init

Use for first-run setup and project bootstrap only.

- Create or update `.specs/project/PROJECT.md` for scope and goals.
- Create or update `.specs/project/ROADMAP.md` for milestones.
- Create or update `.specs/project/STATE.md` for decisions, blockers, and lessons.
- Create or update `.specs/project/HANDOFF.md` for pause/resume context.
- Leave historical artifacts untouched.
