# code-review

Executes code review using the review axes framework. Loads context from project code and provides multi-dimensional quality assessment and feedback.

## Triggers

**English:** `/review`, `code review`, `review this`, `check quality`

**Português:** `revisa isso`

## Install

```bash
node install.js --skill code-review
```

For the full meta-skill or all skills:
```bash
node install.js              # meta-skill only
node install.js --all        # all skills
```
