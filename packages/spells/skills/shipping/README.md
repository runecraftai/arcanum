# shipping

Executes shipping workflow with release checklist and versioning. Loads context from project version and changelog to manage releases and versioning.

## Triggers

**English:** `/ship`, `release`, `publish`, `ship it`

**Português:** `vamos fazer release`, `versiona`

## Install

```bash
node install.js --skill shipping
```

For the full meta-skill or all skills:
```bash
node install.js              # meta-skill only
node install.js --all        # all skills
```
