# code-simplification

Executes code simplification using simplification patterns and refactoring. Loads context from project code to reduce complexity and improve maintainability.

## Triggers

**English:** `/simplify`, `refactor`, `simplify this`, `reduce complexity`

**Português:** `simplifica`, `refatora`

## Install

```bash
node install.js --skill code-simplification
```

For the full meta-skill or all skills:
```bash
node install.js              # meta-skill only
node install.js --all        # all skills
```
