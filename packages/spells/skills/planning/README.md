# planning

Creates vertical slicing plan with task breakdown and dependency ordering. Loads context from project specifications and produces organized task execution plan.

## Triggers

**English:** `/plan`, `plan this`, `break into tasks`, `design the approach`

**Português:** `vamos planejar`, `quebra em tarefas`

## Install

```bash
node install.js --skill planning
```

For the full meta-skill or all skills:
```bash
node install.js              # meta-skill only
node install.js --all        # all skills
```
