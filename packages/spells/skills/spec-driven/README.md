# spec-driven

A 7-phase meta-skill that orchestrates the complete development lifecycle: SPEC, PLAN, BUILD, TEST, REVIEW, SIMPLIFY, SHIP. Routes user triggers to individual phase handlers.

## Triggers

**English:** `/spec`, `specify`, `write spec`, `/plan`, `plan this`, `/build`, `build this`, `/test`, `test this`, `/review`, `code review`, `/simplify`, `refactor`, `/ship`, `release`

**Português:** `vamos especificar`, `preciso de um spec`, `vamos planejar`, `quebra em tarefas`, `vamos construir`, `implementar`, `vamos testar`, `teste isso`, `revisa isso`, `simplifica`, `refatora`, `vamos fazer release`, `versiona`

## Install

For the meta-skill (recommended):
```bash
node install.js
```

For all skills including this one:
```bash
node install.js --all
```

Or just this skill:
```bash
node install.js --skill spec-driven
```

## About

The spec-driven meta-skill is the master orchestrator that routes all 7 development lifecycle phases through a single entry point. It includes 26 supporting reference files for each phase.
