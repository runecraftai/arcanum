# test-verification

Executes test verification using the prove-it pattern. Loads context from project tests and code to ensure comprehensive coverage and behavior verification.

## Triggers

**English:** `/test`, `test this`, `verify`, `prove it works`

**Português:** `vamos testar`, `teste isso`

## Install

```bash
node install.js --skill test-verification
```

For the full meta-skill or all skills:
```bash
node install.js              # meta-skill only
node install.js --all        # all skills
```
