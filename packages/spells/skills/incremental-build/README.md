# incremental-build

Executes incremental build cycle with task execution and verification. Loads context from project tasks and build state, executing tasks sequentially.

## Triggers

**English:** `/build`, `build this`, `implement`, `execute tasks`

**Português:** `vamos construir`, `implementar`

## Install

```bash
node install.js --skill incremental-build
```

For the full meta-skill or all skills:
```bash
node install.js              # meta-skill only
node install.js --all        # all skills
```
